# Palindrome

Given a string, check it is a palindrome.

## Examples

### madam

```
true
```

### cat

```
false
```

### Madam, I'm Adam

```
true
```

### 11,411

```
true
```

### 11,c4ac11

```
false
```

### Never odd or even

```
true
```
