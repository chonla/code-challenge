# Get Changes

Given an amount to pay and amount of payment in cash, find the changes.

## Condition

* Cash unit 1000, 500, 100, 50, 20, 10, 5, 1, 0.5, 0.25
* Use the most value unit first

## Example

| Amount | Paid |
| - | - |
| 20 | 1000 |

```
[[ 500, 1], [100, 4], [50, 1], [20, 1], [10, 1]]
```
