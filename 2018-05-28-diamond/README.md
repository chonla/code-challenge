# Draw a diamond

Given n, draw a diamon with height of (2n - 1). Upper half of diamon has height of n and lower half has n - 1.

## For example

### n = 1

```
*
```

### n = 5

```
    *
   ***
  *****
 *******
*********
 *******
  *****
   ***
    *
```
