# Braces Match

Given a math expression, check if braces are put properly.

## Conditions

* Braces: ( ), [ ], { }
* Operators: + - * /
* Operand may be not a single digit

## Examples

### (5+6)∗(7+8)/(4+3) 

```
true
```

### [(5+6)∗(7+8)/(4+3) 

```
false
```

### (5+6[)∗(7+8])/(4+3)

```
false
```

### (45+6)∗(7+8)/(4+3) 

```
true
```
